# To-do-application

## Brief description

This project is a simple to-do application. 

In this application, a user should be able to:
- sign up with a valid email and password,
- sign in with said email and password, 
- create a new to-do item,
- edit to make changes to and update a list item, 
- view all to-do items created,
- delete an item created.

This app is user specific, as such any user who logs in can see only tasks that the said user has created. Tasks created by other users are not available to anyone else aside the user who created the task.

Finally, once a user is done using the application, this user is able to logout of the application, thereby protecting the contents of his/her task list.
